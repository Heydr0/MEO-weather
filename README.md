# MEO-weather

**Автор:** Рахман Платон Евгеньевич

## Описание программы

Простое приложение для ведения дневника погоды. Позволяет добавлять записи с датой, температурой, описанием и осадками. Есть фильтрация по дате и температуре. Данные сохраняются в файл JSON.

## Как запустить

1. Установите Python 3.14
   
3. Скачайте файл main(2).py
   
4. Запустите приложение:
   ``
   python  main (2).py 
   ``
5. Используйте интерфейс для добавления и фильтрации записей.
